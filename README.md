# React Task App

A car rental UI project built with `React + Vite`, including Home, Favorites, Car Detail, and Payment pages.

## Tech Stack

- React
- React Router DOM
- SCSS
- Vite

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Routes

- `/` — Home
- `/favorites` — Favorites
- `/car/:id` — Car Detail
- `/payment/:id` — Payment

## Main Features

- Header search by car name
- Left filter sidebar (type, capacity, price)
- Add/remove favorites
- Car detail page for selected car
- Payment page sections:
  - Billing Info
  - Rental Info
  - Payment Method
  - Confirmation
  - Rental Summary
